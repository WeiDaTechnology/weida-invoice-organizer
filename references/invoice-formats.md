# 中国发票格式参考

## 增值税电子普通发票

### 标准字段位置

```
发票号码：20 位数字
开票日期：YYYY 年 MM 月 DD 日
购买方名称：在"购买方"区域
销售方名称：在"销售方"区域
价税合计（小写）：¥XXX.XX
项目名称：*税收分类编码*项目名称
```

### 文本结构示例

```
电子发票（普通发票）
发票号码：26317907100200441189
开票日期：2026 年 02 月 10 日

购买方
名称：上海维搭信息科技有限公司
统一社会信用代码/纳税人识别号：91310115MAD13MQM35

销售方
名称：上海城投环城高速建设发展有限公司
统一社会信用代码/纳税人识别号：91310101591633298T

项目名称    金额    税率    税额
*经营租赁*通行费    0.46    3%    0.01

价税合计（大写）肆角柒分
（小写）¥0.47
```

### 正则表达式模式

```python
# 发票号码
r'发票号码 [:：]\s*(\d{20})'

# 开票日期
r'开票日期 [:：]\s*(\d{4}) 年 (\d{1,2}) 月 (\d{1,2}) 日'

# 金额（小写）
r'\(小写\)\s*¥\s*([\d.]+)'

# 购买方名称
r'购买方.*?名称 [:：]\s*([^\n 售方]+)'

# 销售方名称
r'销售方.*?名称 [:：]\s*([^\n]+)'

# 项目名称
r'\*([^*]+)\*([^\n]+)'
```

## 常见发票类型

### 1. 通行费发票

特点：
- 项目名称包含"通行费"
- 有车牌号、通行日期等信息
- 金额通常较小

### 2. 餐饮发票

特点：
- 项目名称为"*餐饮服务*餐费"
- 销售方多为餐饮公司
- 金额不定

### 3. 办公用品发票

特点：
- 项目名称为"*办公用品*"
- 可能有多个商品明细
- 通常有购物清单

## 提取优先级

1. **文字 PDF** - 直接用 pdfplumber 提取（快速、准确）
2. **扫描件** - 需要 OCR（PaddleOCR）
3. **图片格式** - 需要 OCR + 发票定位
